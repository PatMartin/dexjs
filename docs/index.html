dex.js

dex.js is a javascript framework for visualizing data.

Available Charts

  • Area Charts
  • Stacked Area Charts
  • Line Charts
  • Bar Charts
  • Stacked Bar Charts
  • Pie Charts
  • Donut Charts
  • Scatterplots

Go down for live examples or right to continue

C3 Line Chart

// Instantiate the line chart given dataset ncsv
var linechart = new dex.charts.c3.LineChart({
  'parent': '#C3LineChart', // The node this chart will attach to.
  'csv': ncsv // The data
});
// Render the chart
linechart.render();
