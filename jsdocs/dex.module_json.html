<inner> toCsv(json, header)

Converts JSON and a header to a CSV file. It is used for parallel coordinate brush events where the selected brush must be published to events as a csv. For example, given: json = [ { A: 1, B: 3, C: 5, D: 7 }, { A: 2, B: 4, C: 6, D: 8 } ]; header = [ 'A', 'B', 'C', 'D' ]; This will return a csv where: csv = { header: [ 'A', 'B', 'C', 'D' ], data [[ 1, 4, 5, 7 ], [ 2, 4, 6, 8 ]];
