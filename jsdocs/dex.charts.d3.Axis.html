Class: Axis

Axis

This class constructs a d3 Axis.

new Axis(userConfig)

Parameters:
Name Type Description
userConfig object A user supplied configuration object which will override the defaults.
Properties
Name Type Argument Default Description
parent string The parent node to which this Axis will be attached. Ex: #MyParent will attach to a node with an id = "MyParent".
id string <optional>
 Axis The id of this axis.
class string <optional>
 Axis The class of this axis.
csv csv The user's CSV data.
margin margin The margin data.
Properties
Name Type Argument Default Description
left integer <optional>
 25 The number of pixels to allocate to the left margin.
right integer <optional>
 25 The number of pixels to allocate to the right margin.
top integer <optional>
 0 The number of pixels to allocate the top margin.
bottom integer <optional>
 0 The number of pixels to allocate the bottom margin.
transform string <optional>
 translate(0,2) A SVG transform string. More information can be found in the W3C SVG 1.1 Specification.
column integer <optional>
 0 The column within the supplied CSV to use to generate the Axis.
axis d3axis_spec <optional>
 A D3 axis specification.
title d3text <optional>
 A D3 text specification for the title of this axis.
Implements:
  • dex/component

charts/d3/Axis.js
Example
var axis = dex.charts.d3.Axis({
  'parent' : "#MyAxisContainer",
  'id'     : "MyAxisId"
  'csv'    : { header : [ "X", "Y", "Z" ],
               data   : [[ 1, 2, 3 ], [4, 5, 6], [7, 8, 9]]}
});
